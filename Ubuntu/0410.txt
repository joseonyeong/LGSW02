[디렉토리 정리] # 필수
/bin : 명령어의 실행 파일이 배치되어 있는 디렉터리
	환경설정 시 bin을 기준으로 잡음
/dev : 디바이스 파일이 배치, 디스크, 키보드 하드웨어
	시스템 장치 파일들
/etc : 리눅스에서 돌아가는 다양한 애플리케이션 설정 파일
	시스템 설정 파일들 (네트워크, 사용자 설정 등)
	리눅스 자체의 설정 파일
/home : 사용자 작업 공간
	일반 사용자들의 홈 디렉토리
	다양한 계정 존재 가능
/sbin : 실행파일, 관리자용 명령어가 포함됨 (shudown 등)
/tmp : 임시폴더
	어떤 os는 정기적으로 tmp내 파일 삭제함.
/usr : 사용자가 설치한 애플리케이션 실행 파일, 문서, 라이브러리
	/bin, /sbin, /etc 루트 디렉터리와 구조가 비슷함
	(위의 이름과 비슷한 하위 디렉토리가 생김)
/var : 로그파일, 메일, 캐시 등 파일이 만들어짐

[절대경로 vs 상대경로]
